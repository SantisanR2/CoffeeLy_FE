Nuevo lote

Fecha: es la fecha y hora a la que empieza la siembra del lote.

Distancia: es la distancia que se deja entre las semillas a la hora de sembrar, medida en metros.

Profundidad: es la profundidad a la que se siembran las semillas, medida en cm.

Inclinación: es el ángulo que se le buscará dar al cultivo, medida en grados.

El tipo de semilla se refiere a la variedad de café a sembrar.
